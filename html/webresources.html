  1. HTML

    Summary: On W3Schools for HTML one can go through the step-by-step process of creating a html file website.

    W3Schools is a free website that walks you through using html to build a website. Giving information
    that is walkable for even beginners or people that need a refresher in the syntax of the code.
    There are coding games, chapter quizzes, and exercises that you can use to test the knowledge you just
    learned going through the material and resources that it gives.

    Rating: 5 out of 5 stars ★ ★ ★ ★ ★

  2. CSS

    Summary: W3Schools for CSS is just as good as the HTML resource.

    W3Schools for CSS is the ideal resource for beginners to start out learning about web development. There is a
    reason I have this as a resource for 3 different portions of the website. This is perfect and has taught me and countless
    others how to begin coding in various languages. It is quick to refer back to if you forget something and fast to navigate
    through to find what you need.

    Rating: 5 out of 5 stars ★ ★ ★ ★ ★

  3. Bootstrap 5

    Summary: Bootstrap 5 throught W3Schools is handy and should be used when starting out with web development.

    Bootstrap is a little more involved than just using CSS. This is more for people that already know CSS
    and want to expand their knowledge with more developed way to stylize your website. These three resources on W3Schools can be
    a launching point for newwer web developers. It is a great resource for those that know more and just need a refresher on the
    subjects. Or if you need to look back at certain parts to make sure the syntax is correct.

    Rating: 5 out of 5 stars ★ ★ ★ ★ ★

  4. Color Picker

    Summary: This website is easy to use and very direct to what you need.

    Color codes are numerous and impossible to memorize all colors that can be used on a website. This website
    will let you pick the exact shade you need for your website or find new ones that you didn't think you wanted. It is
    easy to use and you can just copy and paste the code to you CSS. Since this is a website dedicated to finding the right
    color for you and being able to search through the spectrum of the color wheel, it can be overwhelming. However, this
    website breaks it up into a smaller spectrum that is more easily searched through.

    Rating: 5 out of 5 stars ★ ★ ★ ★ ★

  5. Free Code Camp

    Summary: Free Code Camp is a good resource for people who are limited on funds but want to learn about web development.

    Free Code Camp is more structured in the teaching of software development than W3Schools. There are
    certifications that you can receive through this website which could help you in landing a job in the career you are
    looking for. You do have to sign up for the website with your email. Some may not want to do that and prefer using
    W3Schools for the learning. However, Free Code Camp will go deeper in the understanding of software and web development.
    Once you have learned all you can on W3Schools you can go to this website and breeze right through the majority of it as well
    as getting a handle on the more advanced aspects that W3Schools is lacking.

    Rating: 4 out of 5 stars ★ ★ ★ ★ ☆
